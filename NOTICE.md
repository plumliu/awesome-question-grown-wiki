# Notice

`question-grown-wiki` is an independent, trimmed skill set for Codex + Obsidian.

It is based on the public `Ar9av/obsidian-wiki` pattern and adapts ideas from:

- `incremental-zettelkasten`: questions as review units
- `joshylchen/zettelkasten`: Capture -> Explain -> Question -> Refine -> Connect
- `arturseo-geo/llm-knowledge-base`: gaps, review queues, learning-layer maintenance

Those reference projects are not vendored in this repository. Keep cloned reference
repositories and personal Obsidian vaults out of public releases.
