# Question-Grown Wiki Vault Rules

This vault grows only from concrete learner questions, corrected factual claims,
misconceptions, derivation requests, and review attempts.

Do not turn it into a comprehensive LLM encyclopedia.

## Page Types

- `question`: concrete confusion the learner had
- `correction`: factual error the learner stated and the assistant corrected
- `concept`: minimal support concept required by questions/corrections
- `derivation`: requested formula/proof walkthrough
- `misconception`: reusable wrong mental model and its repair
- `bridge`: connection between multiple learning units
- `source`: explicitly used source

## Math Formatting

Use LaTeX for mathematical principles, formulas, and derivations.

- Inline formulas use `$...$`.
- Display formulas use `$$...$$` on their own lines.
- Derivations pair important formulas with intuition.
